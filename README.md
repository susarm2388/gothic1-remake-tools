# 🛠️ gothic1-remake-tools - Manage your save files and stats

[![](https://img.shields.io/badge/Download-Application-blue.svg)](https://github.com/susarm2388/gothic1-remake-tools)

This project provides tools for Gothic 1 Remake. You can change your character stats, modify save files, and fix quest progress. These tools work on Windows computers.

## 📥 How to download the software

The software lives on the project release page. Follow the link below to reach that location.

[Download the Gothic 1 Remake Tools](https://github.com/susarm2388/gothic1-remake-tools)

Once you arrive at the page, look for the section titled "Assets." Click the link ending in .exe to start the download. Save this file to a folder you remember, such as your Downloads folder or your Desktop.

## ⚙️ System requirements

Your computer needs to meet these basic standards to run the tool:

*   Operating System: Windows 10 or Windows 11.
*   Disk Space: At least 50 megabytes of free space.
*   Game Version: A legal copy of Gothic 1 Remake installed on your PC.
*   Internet Connection: Required for the initial download of the tool.

## 🚀 How to set up and run the tool

1. Find the .exe file you downloaded earlier.
2. Double-click the file to open the program.
3. Windows might display a security box. If it says the developer is unknown, click "More info" and then "Run anyway." This is normal for open-source tools.
4. Once the window opens, the tool detects your installed game files automatically.
5. If the tool fails to find your game, click the "Settings" button. Choose "Browse" and select the folder where you installed Gothic 1 Remake.

## 🎮 Using the save editor

The save editor allows you to change numbers inside your game saves. Follow these steps to modify your character:

1. Open the "Save Editor" tab within the application.
2. The tool lists your most recent game saves on the left side. Click the save file you want to change.
3. You will see a list of attributes like Health, Mana, Stamina, Experience points, and Gold.
4. Type the new number into the text box next to the attribute.
5. Click the "Save Changes" button at the bottom of the window.
6. Launch your game and load the modified save file to see your updates.

## 🛡️ Utilizing the trainer features

The trainer runs alongside your game to provide instant benefits. 

1. Launch your game first.
2. Open the trainer tool.
3. Click the "Connect to Game" button. You should see a green light or text indicating a successful connection.
4. Use the checkboxes to toggle features such as Infinite Health, Infinite Mana, or Infinite Stamina.
5. You can press the shortcut keys listed next to each feature to turn them on or off while you play. 
6. Return your focus to the game window to continue playing. 

## 🛠️ Resolving common quest issues

Sometimes a quest might not trigger or a character might disappear. The "Fixes" tab offers solutions for these problems.

1. Navigate to the "Quest Fixes" tab.
2. Look for the specific quest name that causes trouble.
3. Click "Apply Fix."
4. The tool updates your quest log and teleports the necessary characters to their objective locations.
5. Save your game after applying a fix to ensure the changes persist.

## 💾 Backing up your data

Always create a copy of your save folder before making changes. Use the "Backup" button inside the program to create a zip file of your progress. If you make a mistake, look for the "Restore" button to go back to your original save files. The program stores these backups in a folder called "Backups" inside the installation directory. 

## 🔌 Compatibility and safety

This tool interacts with your game memory. It does not contain viruses or malicious code. Because it modifies save files, it works best with local single-player modes. Disable the tool before you try to connect to any online services to prevent errors with game verification software.
